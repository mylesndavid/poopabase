# poop studio

Open [Outerbase Studio](https://outerbase.com) on your poopabase database for visual exploration, querying, and management.

## Usage

```bash
poop studio [options]
```

## Options

| Option | Description | Default |
|--------|-------------|---------|
| `--db <path>` | Path to the database file | `.poop.db` |

## Examples

```bash
# Open studio for the default database
poop studio

# Open studio for a specific database
poop studio --db ./data/myapp.poop.db
```
